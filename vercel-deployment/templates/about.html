📚 Acerca de este proyecto

Este proyecto demuestra cómo crear y desplegar una aplicación web simple usando tecnologías modernas.

🔧 Tecnologías utilizadas:

  • Python 3.x - Lenguaje de programación
  • Flask 2.3.3 - Framework web ligero
  • Jinja2 3.1.2 - Motor de plantillas
  • HTML5 & CSS3 - Frontend moderno
  • Vercel - Hosting gratuito
  • GitHub - Control de versiones

📁 Estructura del proyecto:

vercel-deployment/
├── app.py              # Aplicación Flask principal
├── requirements.txt    # Dependencias de Python
├── vercel.json        # Configuración de Vercel
├── templates/         # Plantillas Jinja2
│   ├── base.html      # Plantilla base
│   ├── index.html     # Página principal
│   ├── about.html     # Esta página
│   └── greet.html     # Página de saludo
└── README.md          # Documentación

🚀 Proceso de despliegue:

  1. Crear repositorio en GitHub
  2. Subir el código al repositorio
  3. Conectar Vercel con GitHub
  4. ¡La aplicación se despliega automáticamente!

💡 ¿Por qué este stack?

Flask es perfecto para proyectos pequeños y medianos, mientras que Jinja2 permite crear plantillas dinámicas de forma elegante. Vercel ofrece hosting gratuito con despliegue automático desde GitHub, ideal para proyectos educativos y prototipos.
